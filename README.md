#web后台管理系统 
* 支持多种终端访问：手机、PAD、PC
* 包含用户管理、组织管理、角色管理、菜单管理、日志管理、报表管理等模块
* 各模块支持上传、下载、分页等功能
* 前端基于Bootstrap、JQuery、FreeMarker等开发，后端基于SpringBoot、Mybatis、Shiro等开发
